======
eafsed
======

A ``sed`` for XML files in the Elan Annotation Format.

Description
===========

Search and replace on specific tiers (as identified by their
``LINGUISTIC_TYPE_REF``) of ``.eaf`` files.

Usage
=====

See ``eafsed --help``.

Installation
============

.. code:: bash

   $ pip3 install git+https://github.com/dlukes/eafsed
